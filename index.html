Editor’s Guide: how to update this page (no code)

  1. Change colors/fonts: at the top of this file in :root, edit --brand-primary, --brand-secondary, --brand-accent. Fonts are loaded from Google Fonts—swap names in the <link> and CSS font-family lists.
  2. Update dates/capacity/price: find the “Upcoming cohort” section and edit the text. Also update the JSON‑LD startDate and offers.price if needed.
  3. Connect email tool (Mailchimp/ConvertKit): replace the forms (#lead and #signup-form) with your provider’s embed form code. Keep the same fields: name, email, role, goal. Add tag InnovateWithOma/Waitlist.
  4. Add images: replace the placeholder src paths (hero and screenshots). Use descriptive alt text. Compress to .webp.
  5. Add FAQs/Testimonials/Blog: duplicate the existing items now. If you move to a CMS (Wix/Webflow/Framer/WordPress), create collections with fields listed in the spec and bind these sections to them.
  6. Analytics: paste your GA4 snippet and replace G-XXXX. Optional: add Meta Pixel.
  7. Publish fast: a) In Wix/Webflow/Framer—create a page and paste sections using their visual editor; or b) For a quick static site, upload this single file as index.html to Netlify/Cloudflare Pages/GitHub Pages.
